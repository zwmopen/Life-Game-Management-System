# 商品分类标准

## 1. 分类定义

### 1.1 数码类
- **定义**：特指电子数码产品及其配件
- **包含范围**：
  - 智能手机、平板电脑、笔记本电脑等移动设备
  - 耳机、音箱、摄像头等数码配件
  - 键盘、鼠标等电脑外设
  - 智能手表、智能手环等可穿戴设备（除鞋类）
  - 摄影摄像设备
- **排除范围**：
  - 穿戴于人体的服饰鞋履类商品
  - 家用大型电器
  - 家居生活用品

### 1.2 家居类
- **定义**：家庭生活中使用的各类用品
- **包含范围**：
  - 家具（沙发、床、桌子等）
  - 家用电器（扫地机器人、洗衣机、冰箱等）
  - 厨房用品（锅碗瓢盆、厨房电器等）
  - 床上用品（枕头、被子等）
  - 人体工学相关产品（人体工学椅、人体工学鼠标垫等）
  - 家居装饰品
- **排除范围**：
  - 穿戴于人体的服饰鞋履类商品
  - 纯电子数码产品

### 1.3 个人穿搭类（服装类）
- **定义**：穿戴于人体的服饰鞋履类商品
- **包含范围**：
  - 衣服、裤子、裙子等服装
  - 鞋子、袜子等鞋履
  - 帽子、围巾、手套等配饰
  - 手表、项链、耳环等饰品
- **排除范围**：
  - 电子数码产品
  - 家居生活用品

### 1.4 其他分类
- **饮食类**：各类食品、饮料
- **娱乐类**：娱乐活动、游戏、休闲体验
- **服务类**：各类服务（理发、按摩、学习课程等）
- **票务类**：各类门票、车票、机票
- **会员类**：各类会员服务
- **充值类**：话费充值、云存储等充值服务
- **房产类**：房屋、公寓等不动产
- **车辆类**：汽车、摩托车等交通工具
- **运动类**：运动器材、运动服装（归属于服装类）
- **礼品类**：各类礼品
- **其他类**：无法归类到以上分类的商品

## 2. 分类流程

1. **判断是否为电子数码产品**：如果是，归为数码类
2. **判断是否为穿戴于人体的服饰鞋履类商品**：如果是，归为服装类
3. **判断是否为家庭生活中使用的用品**：如果是，归为家居类
4. **其他情况**：根据商品属性归为其他相应分类

## 3. 分类示例

| 商品名称 | 正确分类 | 错误分类 | 原因 |
|---------|---------|---------|------|
| 智能手机 | 数码类 | 家居类 | 属于电子数码产品 |
| 人体工学椅 | 家居类 | 数码类 | 属于家庭生活中使用的人体工学产品 |
| 鞋子 | 服装类 | 数码类 | 属于穿戴于人体的鞋履类商品 |
| 扫地机器人 | 家居类 | 数码类 | 属于家庭生活中使用的电器 |
| 防蓝光眼镜 | 数码类 | 服装类 | 属于数码配件 |
| 乳胶枕头 | 家居类 | 服装类 | 属于家庭床上用品 |

## 4. 分类管理

1. 新增商品时，需按照本标准进行分类
2. 定期检查现有商品分类，确保符合本标准
3. 如有分类争议，需由产品团队讨论决定
4. 本标准将根据业务发展情况定期更新
