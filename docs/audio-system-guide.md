# 音频管理系统使用指南

## 目录结构

音频文件应放置在 `public/audio/` 目录下的相应子目录中：

```
public/
└── audio/
    ├── bgm/                 # 背景音乐
    ├── battle/              # 战斗音效
    ├── dice/                # 掷骰子音效
    ├── pomodoro/            # 番茄钟音效
    │   └── bgm/             # 番茄钟背景音乐
    └── sfx/                 # 通用音效
```

## 使用方法

### 添加新音频文件

1. 将音频文件（支持 .mp3、.wav、.ogg、.m4a 格式）复制到相应的目录中
2. 系统会在下次启动时自动扫描并识别新添加的音频文件
3. 音频文件名将被用作显示名称（会自动格式化为可读格式）

### 推荐命名方式

- 使用有意义的文件名，如 `rain-sound.mp3`、`focus-music-1.mp3`
- 避免使用特殊字符和空格
- 可以使用数字后缀来区分类似音频，如 `bgm-1.mp3`, `bgm-2.mp3`

### 音频分类

- **背景音乐 (bgm)**: 用于游戏背景播放的长音频文件
- **番茄钟背景音乐 (pomodoro/bgm)**: 专门用于番茄钟功能的背景音乐
- **番茄钟音效 (pomodoro)**: 用于番茄钟开始、结束等事件的短音效
- **战斗音效 (battle)**: 战斗相关的音效
- **掷骰子音效 (dice)**: 掷骰子时播放的音效
- **通用音效 (sfx)**: 其他通用音效

## 注意事项

- 音频文件会按需加载，以提高性能
- 系统会自动检测目录中的音频文件，无需手动配置
- 文件名将自动转换为用户友好的显示名称
