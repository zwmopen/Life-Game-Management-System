# 音频管理库使用说明

## 目录结构

音频文件应放置在 `public/audio/` 目录下，按功能分类存放：

```
public/
└── audio/
    ├── battle/     # 战斗音效
    ├── bgm/        # 背景音乐
    ├── dice/       # 掷骰子音效
    ├── pomodoro/   # 番茄钟音效
    └── sfx/        # 通用音效
```

## 使用方法

### 1. 添加音频文件

将音频文件（MP3、WAV、OGG 格式）放入对应的分类文件夹中。

### 2. 文件命名建议

- 使用有意义的文件名，如 `forest-background.mp3`、`dice-roll.wav`
- 文件名将作为音频的显示名称

### 3. 在代码中使用

```typescript
import audioManager from '../utils/audioManager';

// 初始化音频管理器
await audioManager.initialize();

// 获取所有背景音乐
const bgmFiles = audioManager.getBackgroundMusic();

// 播放音频
const audio = audioManager.playAudio('/audio/bgm/forest.mp3', 0.7);
```

## 音频管理面板

系统提供了可视化音频管理面板，可在设置中心找到。通过该面板可以：

- 浏览所有音频文件
- 预览音频效果
- 选择当前使用的音频
- 调整播放音量

## 注意事项

1. 确保音频文件格式为 MP3、WAV 或 OGG
2. 音频文件大小不宜过大，以免影响加载速度
3. 建议音频时长适中，避免长时间音频占用资源
4. 文件名中避免使用特殊字符
