# 人生游戏化系统-命运骰子任务事件开发文档
## 文档版本
V1.0
## 编写日期
2026年01月12日
## 核心目标
在人生游戏化系统**任务管理模块-副本任务管理子模块**中，搭建「命运骰子」功能的完整命运事件任务池，包含健康微行动、效率任务、休闲小奖励三类任务，实现**随机抽取、分类管理、7天去重防重复**的核心逻辑，同时支持任务信息的灵活编辑。

## 一、开发约束
1. **模块归属**：所有任务事件统一录入「副本任务管理-命运事件」模块，按“健康微行动”“效率任务”“休闲小奖励”分类标签归档，清空所有任务的“未分类”标签。
2. **功能联动**
    - 任务池与命运骰子随机抽取逻辑绑定，抽取占比为：健康微行动（4面）、效率任务（5面）、休闲小奖励（3面）。
    - 系统需记录用户7天内抽取的历史任务，实现抽取时的去重逻辑，避免重复获取。
3. **数值规则**：所有任务的**预估时间消耗（分钟）** 与**金币奖励（个）** 数值完全一致，无偏差。
4. **扩展支持**：保留所有任务事件的编辑入口，支持修改「任务名称、完成标准、时间消耗、金币奖励、分类归属」，修改后实时生效。

## 三、最终验收标准
1. 三类任务全部完整录入「命运事件」模块，分类标签分配准确，无“未分类”任务；
2. 所有任务的预估时间消耗与金币奖励数值完全一致，无任何数值偏差；
3. 命运骰子抽取功能正常，可按「健康4面、效率5面、休闲3面」的占比随机获取任务，且7天内不会抽取到重复任务；
4. 任务编辑入口功能正常，支持修改任务名称、完成标准、时间消耗、金币奖励、分类归属，修改后系统实时生效。
