# 音频资源管理说明

本目录包含应用程序中使用的所有音频资源。以下是各子目录的详细说明：

## 目录结构

### `/bgm` - 背景音乐
- **用途**：通用背景音乐
- **使用场景**：系统通用背景音乐播放
- **支持格式**：`.mp3`, `.wav`, `.ogg`, `.m4a`
- **文件命名**：建议使用有意义的名称，如 `forest.mp3`, `ocean-waves.mp3`
- **说明文件**：`README.txt` 提供详细说明

### `/pomodoro/bgm` - 番茄钟背景音乐
- **用途**：番茄钟功能专用背景音乐
- **使用场景**：番茄钟专注/休息时段的背景音乐
- **支持格式**：`.mp3`, `.wav`, `.ogg`, `.m4a`
- **文件命名**：建议使用描述性名称，如 `focus-music.mp3`, `relaxing-sound.mp3`
- **说明文件**：`README.txt` 提供详细说明

### `/pomodoro` - 番茄钟音效
- **用途**：番茄钟功能专用提示音效
- **使用场景**：开始/结束提示音、警告提示音等
- **支持格式**：`.mp3`, `.wav`, `.ogg`, `.m4a`
- **文件命名**：建议使用功能相关名称，如 `start.mp3`, `end.mp3`, `warning.mp3`
- **说明文件**：`README.txt` 提供详细说明

### `/battle` - 战斗音效
- **用途**：战斗相关音效
- **使用场景**：战斗系统中的各种音效
- **支持格式**：`.mp3`, `.wav`, `.ogg`, `.m4a`
- **文件命名**：建议使用战斗相关名称，如 `victory.mp3`, `defeat.mp3`, `alert.mp3`
- **说明文件**：`README.txt` 提供详细说明

### `/dice` - 骰子音效
- **用途**：骰子功能音效
- **使用场景**：掷骰子时的音效
- **支持格式**：`.mp3`, `.wav`, `.ogg`, `.m4a`
- **文件命名**：建议使用骰子相关名称，如 `normal.mp3`, `special.mp3`, `magic.mp3`
- **说明文件**：`README.txt` 提供详细说明

### `/sfx` - 通用音效
- **用途**：系统通用音效
- **使用场景**：按钮点击、通知提示等
- **支持格式**：`.mp3`, `.wav`, `.ogg`, `.m4a`
- **文件命名**：建议使用功能相关名称，如 `click.mp3`, `notification.mp3`, `achievement.mp3`
- **说明文件**：`README.txt` 提供详细说明

### `/sounds` - 额外音效
- **用途**：额外系统音效
- **使用场景**：系统中其他功能的音效
- **支持格式**：`.mp3`, `.wav`, `.ogg`, `.m4a`
- **文件命名**：建议使用功能相关名称
- **说明文件**：`README.txt` 提供详细说明

## 使用说明

1. **添加音频文件**：将音频文件放入对应的目录中，系统会自动扫描并识别
2. **文件格式**：建议使用 MP3 格式以获得最佳兼容性
3. **文件大小**：建议单个文件不超过 5MB，避免影响加载速度
4. **命名规范**：
   - 使用英文或拼音命名
   - 避免使用特殊字符和空格（可用连字符 `-` 代替）
   - 文件名应能反映音频内容
5. **说明文档**：每个子目录都包含 `README.txt` 文件，提供该目录的详细说明

## 注意事项

- 音频文件会根据所在目录自动归类到相应的音频类别
- 新添加的音频文件在重新加载页面后即可在音频管理界面中看到
- 所有音频文件必须放置在正确的目录中才能被系统正确识别和使用
